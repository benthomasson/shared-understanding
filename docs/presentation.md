### Slide 5: The Technology Stack
**[Title: Zero-Installation Integration with Your Existing Tools]**

**Core Components:**
- **Git Repository Foundation**: Version-controlled knowledge base that can be forked, shared, and collaborated on
- **Chronological Entries**: Date-structured documentation (entries/YYYY/MM/DD/)
- **Living Documents**: Evolving summaries and architectural plans (docs/)
- **External Integrations**: Jira, Google Drive, Slack via uvx (zero installation)
- **Visual Analysis**: AI surprisingly good at understanding screenshots and diagrams
- **Memory Persistence**: claude-projects tool for conversation archival

**Skills Integration:**
```bash
# Jira context import
uvx jirahhh show PROJ-123

# Google Drive meeting notes
uvx gcmd export "meeting-notes-url" -o /tmp/

# Slack organizational search  
uvx slacker search "technical decision keywords"

# Conversation archival for long-term memory
claude-projects convert ~/.claude/projects to markdown
```

*"Git-based knowledge sharing with familiar workflows - fork, commit, merge, pull request. No IT overhead, no dependency conflicts, plus visual understanding and persistent memory."*

### Slide 7: Overcoming Human and AI Limitations
**[Title: External Memory for Both Species]**

**Human Limitations (Kahneman's "Thinking Fast and Slow"):**
- Short-term memory constraints and information silos
- **System 1 thinking:** Fast, automatic responses based on pattern recognition
- **Familiarity masquerading as understanding:** We feel we understand because it seems familiar
- Cognitive biases and selective attention during discussions

**AI Limitations:**
- Context window boundaries and conversation compacting
- **Temporal confusion:** Attention mechanisms lose track of dates, weighting old and new information equally
- Probabilistic memory drift and session isolation
- No persistent memory across conversations

**The Solution:** 
- **Markdown as external memory** for both humans and AI
- **Chronological structure (YYYY/MM/DD)** helps AI overcome temporal weighting problems
- **Forces System 2 thinking** - slow, deliberate, analytical processing
- **Conversation archival** using claude-projects tool for long-term memory

*"Writing things down isn't just good practice - it's cognitive enhancement that solves fundamental computational problems for both human and artificial intelligence."*
